Examples are loaded by the GUI from this folder.

Each example uses three files with the same base name:
  name.domain
  name.query
  name.expected

The .expected file contains TAK or NIE.

The GUI starts as a simple IDE for reasoning. Click "Pokaż przykłady" to display this example list.

The final set includes:
  - examples based on the checkpoint theory document,
  - extra TAK/NIE examples,
  - regression examples,
  - 8 scenario examples, including dining philosophers.

User-facing formulas use word operators only:
  not, and, or, implies, iff, true, false
